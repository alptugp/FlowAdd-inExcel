This add-in allows you to pull values from Flow and push the current Excel workbook to your OneDrive

  1. In the corresponding fields at the bottom of this taskpane, enter your Flow account's email, password and name of the project in Flow which contains the values to be pulled and then click Save.
  2. To pull a value from Flow in a cell, type =CONTOSO.FLOW("Value Name") in the cell. The function's input should be the name of the value to be pulled from Flow written inside double quotes.
  3. Press Enter within the cell to only run the function in it or click on Run All Functions below to run all the functions used in the current workbook.
  4. When you want to make a copy of the current workbook and push it as a file to OneDrive, click on Copy and Push Current Workbook.
